# AGENTS.md

이 프로젝트는 **학생부 PDF 분석 웹앱**입니다.

## 프로젝트 특성
- 배포 환경: **GitHub Pages**
- 기본 실행 흐름: `index.html` → `events.js` → `main.js` → `pdfLoader.js`

## 작업 원칙
- **기존 구조 유지**: 파일 분리/호출 순서와 역할을 바꾸지 않습니다.
- **최소 수정**: 요구사항 충족에 필요한 범위만 변경합니다.
- **UI 한국어 유지**: 사용자에게 노출되는 문구는 한국어를 기본으로 유지합니다.
- **먼저 수정 계획 설명**: 실제 코드/문서 수정 전에 변경 계획을 간단히 공유합니다.
